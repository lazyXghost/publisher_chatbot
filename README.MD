## Steps for running rag -:
1. Create .env file in root folder and add the following environment variables
```
OPENAI_API_KEY=<YOUR OPENAI KEY>
```
2. Run the following commands:
```
    pip3 install -r requirements.txt
    python3 rag.py
```
